화면 별 주요 기능

    1. image_picker_view.dart (선택한 이미지 미리보기 화면)

        역할: image_picker.dart에서 사용자가 선택한 이미지들을 미리 보여주는 화면입니다. 사용자가 선택한 이미지들을 한눈에 확인하고, 필요에 따라 선택을 취소하거나, 선택한 이미지를 최종적으로 확정할 수 있는 기능을 제공합니다.

        주요 기능:
            선택된 이미지 목록 표시: image_picker.dart에서 선택한 이미지들이 격자(Grid) 또는 리스트(List) 형태로 표시됩니다.
            이미지 제거: 각 이미지에는 제거 버튼이 있어, 사용자가 원치 않는 이미지를 선택 목록에서 제거할 수 있습니다.
            동영상 길이 표시: 선택된 이미지 중 동영상이 있을 경우, 해당 동영상의 재생 시간을 표시합니다.
            이미지 미리보기: 이미지를 탭하면 해당 이미지를 확대하여 볼 수 있습니다.
            선택 완료: 선택한 이미지들을 최종적으로 확정하고, 이전 화면으로 돌아가 선택된 이미지 목록을 전달합니다.
            선택한 이미지 없을 시: 선택한 이미지가 없을 시 아무것도 표시되지 않습니다.

        사용 시나리오:
            사용자가 image_picker.dart에서 여러 이미지를 선택합니다.
            선택한 이미지들을 확인하기 위해 image_picker_view.dart로 이동합니다.
            image_picker_view.dart에서 선택한 이미지들을 확인하고, 필요에 따라 이미지를 제거합니다.
            선택이 완료되면, 선택된 이미지 목록을 가지고 이전 화면으로 돌아갑니다.

        화면 구성:
            GridView 또는 ListView를 사용하여 선택한 이미지를 표시합니다.
            각 이미지에는 제거 버튼이 있습니다.
            동영상일 경우 동영상 길이 표시가 있습니다.
            선택한 이미지가 없을 시 아무것도 표시되지 않습니다.

    2. image_picker.dart (이미지 선택 화면)

       역할: 기기에 저장된 모든 이미지 및 동영상을 앨범별로 보여주고, 사용자가 원하는 이미지 또는 동영상을 선택할 수 있도록 하는 화면입니다.

       주요 기능:
            앨범 목록 표시: 기기에 저장된 모든 앨범 목록을 표시합니다.
            앨범 선택: 사용자가 앨범을 선택하면, 해당 앨범에 포함된 이미지 및 동영상 목록을 표시합니다.
            이미지 및 동영상 목록 표시: 선택된 앨범에 포함된 이미지 및 동영상 목록을 격자(Grid) 형태로 표시합니다.
            이미지 선택: 사용자가 이미지를 탭하면, 해당 이미지를 선택 목록에 추가하거나 제거합니다.
            전체 화면 보기: 이미지를 탭하면, 해당 이미지를 전체 화면으로 볼 수 있는 full_image_view.dart로 이동합니다.
            동영상 길이 표시: 동영상일 경우, 해당 동영상의 재생 시간을 표시합니다.
            선택 완료: 선택한 이미지들을 최종적으로 확정하고, 이전 화면으로 돌아가 선택된 이미지 목록을 전달합니다.
            선택한 이미지 개수 표시: 선택한 이미지 개수를 표시합니다.
            선택한 이미지 전송: 선택한 이미지를 전송합니다.

        사용 시나리오:

            1.  선택한 이미지 리스트, 선택한 앨범 초기화 X (유지)**
                `image_picker.dart`로 이동했을 때, 이전에 선택했던 이미지 리스트를 그대로 유지합니다.
                `image_picker_view.dart`에서 `image_picker.dart`로 이동할 때, 이전에 선택했던 앨범을 그대로 보여줍니다.
                사용자가 이전에 선택했던 이미지와 앨범을 이어서 사용할 수 있도록 합니다.

            2.  **선택한 이미지 리스트, 선택한 앨범 초기화 O (초기화)**
                `image_picker.dart`로 이동할 때마다, 항상 전체 이미지를 보여주는 앨범을 기본으로 보여줍니다.
                `image_picker.dart`에서 이전에 선택했던 이미지 리스트는 초기화되어, 빈 리스트로 시작합니다.
                사용자가 매번 새로운 이미지 선택을 시작할 수 있도록 합니다.

            사용자가 이미지 또는 동영상을 선택하기 위해 image_picker.dart로 이동합니다.
            image_picker.dart에서 앨범 목록을 확인하고, 원하는 앨범을 선택합니다.
            선택한 앨범의 이미지 및 동영상 목록을 확인하고, 원하는 이미지 또는 동영상을 선택합니다.
            이미지를 탭하여 전체 화면으로 보거나, 동영상 길이를 확인합니다.
            선택이 완료되면, 선택된 이미지 목록을 가지고 이전 화면으로 돌아갑니다.

        화면 구성:
            DropdownButton을 사용하여 앨범 목록을 표시합니다.
            GridView를 사용하여 이미지 및 동영상 목록을 표시합니다.
            각 이미지에는 선택 버튼이 있습니다.
            동영상일 경우 동영상 길이 표시가 있습니다.
            선택한 이미지 개수 표시가 있습니다.
            선택한 이미지 전송 버튼이 있습니다.

    3. full_image_view.dart (전체 화면 이미지 보기 화면)

        역할: image_picker.dart에서 사용자가 선택한 이미지를 전체 화면으로 보여주는 화면입니다. image_picker.dart와 동일하게 이미지 선택 기능을 제공하여, 전체 화면에서 이미지를 확인하면서 선택을 변경할 수 있습니다.

        주요 기능:
            전체 화면 이미지 표시: image_picker.dart에서 선택한 이미지를 전체 화면으로 표시합니다.
            스와이프를 통한 이미지 이동: 좌우 스와이프를 통해 이전/다음 이미지로 이동할 수 있습니다.
            이미지 선택: image_picker.dart와 동일하게 이미지를 선택하거나 선택을 취소할 수 있습니다.
            동영상 길이 표시: 동영상일 경우, 해당 동영상의 재생 시간을 표시합니다.
            선택 완료: 선택한 이미지들을 최종적으로 확정하고, 이전 화면으로 돌아가 선택된 이미지 목록을 전달합니다.
            선택한 이미지 개수 표시: 선택한 이미지 개수를 표시합니다.
            선택한 이미지 전송: 선택한 이미지를 전송합니다.

        사용 시나리오:
            사용자가 image_picker.dart에서 이미지를 탭하여 전체 화면으로 봅니다.
            full_image_view.dart에서 이미지를 전체 화면으로 확인합니다.
            좌우 스와이프를 통해 다른 이미지를 확인합니다.
            필요에 따라 이미지를 선택하거나 선택을 취소합니다.
            선택이 완료되면, 선택된 이미지 목록을 가지고 이전 화면으로 돌아갑니다.

        화면 구성:
            PageView를 사용하여 이미지를 전체 화면으로 표시합니다.
            각 이미지에는 선택 버튼이 있습니다.
            동영상일 경우 동영상 길이 표시가 있습니다.
            선택한 이미지 개수 표시가 있습니다.
            선택한 이미지 전송 버튼이 있습니다.
